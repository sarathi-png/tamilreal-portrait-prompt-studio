# TamilReal • Portrait Prompt Studio

**The only prompt studio that treats your face as a literal pixel stamp.**

Ultra-strict identity-locked prompts for **ChatGPT** and **Gemini** only.  
No DALL·E. No Flux. No face morphing. 200% EFCP (Exact Face Composite Protocol) enforced.

## ✨ Features
- 200% EFCP Nuclear Face Lock (pixel-stamp mode)
- Identity Strength Slider (100%–200%)
- One-click model-optimized variants (ChatGPT / Gemini / General)
- Live length bar + length validation
- Smart Analyzer with Face Fidelity Score
- Powerful Enhancer with EFCP reinforcement
- Saved prompt library with tags
- Batch export (5 location variations)
- 5 beautiful themes (Claymorphism default)

## 🚀 How to Use
1. Open `index.html` in any browser (no installation needed)
2. Upload your reference photo **in ChatGPT or Gemini** (the prompt reminds you)
3. Use the Creator tab to build perfect portraits
4. Click **Generate → ChatGPT Optimized** or **Gemini Optimized**
5. Copy & paste into your AI tool
